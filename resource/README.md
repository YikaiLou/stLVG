# Resources

## SimpleElastix(optional)
If you need SLAT pre-matched function runs with picture input (such as H&E staining), please install `SimpleElastix` here follow its official [documents](https://simpleelastix.readthedocs.io/)

## scglue(optional)
SLAT collaborate with [scglue](https://github.com/gao-lab/GLUE) when align spatial multi-omics data. `scglue` needs gene annotation in gtf format, please download corresponding file depends on species and genome version to `/glue` subfolder.

## R(optional)
`R` is necessary if you want to run benchmark pipeline, we recommend you compile a new `R`, please follow [`env/README.md`](env/README.md) to install `R` and `R` packages.
